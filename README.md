# PAPER — FAKER Portfolio

페이커를 소개하는 멀티 페이지 포트폴리오 웹사이트입니다.

## 페이지 구성

- `index.html` — 메인 및 카테고리
- `about.html` — 페이커 소개
- `career.html` — 활동 기간
- `t1.html` — T1 입단 배경
- `records.html` — 주요 업적과 레전드 판단 영상
- `goal.html` — 페이커의 목표

## 폴더 구조

## 로컬 실행

`index.html` 파일을 브라우저로 실행합니다.

## GitHub Pages 배포

1. GitHub에서 새 저장소를 만듭니다.
2. 이 폴더 안의 파일을 저장소에 업로드합니다.
3. 저장소의 `Settings` → `Pages`로 이동합니다.
4. `Deploy from a branch`를 선택합니다.
5. Branch를 `main`, 폴더를 `/root`로 선택한 뒤 저장합니다.
6. 생성된 GitHub Pages 주소로 접속합니다.
